# ▶️ Como Executar o Projeto

Este projeto utiliza **Graphviz** para gerar a visualização do grafo em formato de imagem.

⚠️ **Atenção:** É necessário ter o **Graphviz instalado no sistema operacional**, além da biblioteca Python `graphviz`.

---

## ✅ Pré-requisitos

- **Python 3.8 ou superior**
- **Graphviz (ferramenta do sistema)**
- **Biblioteca Python `graphviz`**

---

## 🧩 Instalando o Graphviz no Sistema

### 🪟 Windows
1. Baixe o instalador em:  
   👉 https://graphviz.org/download/
2. Instale o Graphviz
3. Marque a opção **Add Graphviz to PATH** (ou adicione manualmente)
4. Reinicie o terminal

### 🐧 Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install graphviz
